# grocery-sales-data--power-bi-dashbord
Designed and developed an interactive business intelligence dashboard  using Power BI Desktop to track and analyze sales performance.

Tech Stack:
- Power BI Desktop — Data visualization
- Excel/CSV — Data source
- DAX — Custom measures and calculations

 Key Features:
- KPI cards for total sales, profit and quantity
- Charts for category, region and time-based analysis
- Dynamic slicers for filtering by date, region and category.
